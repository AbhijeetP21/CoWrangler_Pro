Setup Instructions

- mkdir backend
- mkdir frontend


Setting up the frontend:
- cd frontend
- npx create-react-app cowrangler-ui
- cd cowrangler-ui
- npm install axios bootstrap react-bootstrap ag-grid-community ag-grid-react react-syntax-highlighter @monaco-editor/react lodash react-icons
- mkdir -p src/components src/services


Added the following files to components folder:
- DataPreview.js - For the data grid panel
- SuggestionsPanel.js - For the suggestions list
- CodeEditor.js - For the code preview/editing
- TransformationsHistory.js - For the applied transformations panel
- MainLayout.js - For the overall UI layout


Setting up the backend:
- Go back to root directory
- cd backend
- python -m venv venv (Create and activate virtual environment)
- If you are on Windows, run venv\Scripts\activate. If you are on MacOS or Linux, run source venv/bin/activate
- pip install flask flask-cors pandas numpy scikit-learn pytest black (INSTALLING DEPENDENCIES)
- mkdir cowrangler
- mkdir cowrangler\learners
- # Create empty Python files using New-Item
- New-Item -Path "cowrangler\__init__.py" -ItemType File -Force
- New-Item -Path "cowrangler\data_analysis.py" -ItemType File -Force
- New-Item -Path "cowrangler\suggestion_engine.py" -ItemType File -Force
- New-Item -Path "cowrangler\code_generator.py" -ItemType File -Force
- New-Item -Path "cowrangler\learners\__init__.py" -ItemType File -Force
- New-Item -Path "cowrangler\learners\drop_learner.py" -ItemType File -Force
- New-Item -Path "cowrangler\learners\split_learner.py" -ItemType File -Force
- New-Item -Path "cowrangler\learners\typecast_learner.py" -ItemType File -Force
- New-Item -Path "cowrangler\learners\missing_values_learner.py" -ItemType File -Force
- New-Item -Path "cowrangler\learners\encoding_learner.py" -ItemType File -Force
- # Create the main Flask application file
- New-Item -Path "app.py" -ItemType File -Force
